# DermGAN: Controlled GAN-Based Data Augmentation for Skin Lesion Classification

A reproducible experimental framework for evaluating GAN-based synthetic augmentation in imbalanced medical image classification.

---

## 1. Overview

Class imbalance is a critical challenge in medical image classification, particularly in dermoscopic skin lesion datasets. Rare lesion types often contain significantly fewer samples than common classes, leading to:

- Model bias toward majority classes  
- Poor recall for rare diseases  
- Reduced clinical reliability  

**DermGAN** proposes and evaluates a controlled GAN-based data augmentation framework to improve classification performance on imbalanced skin lesion datasets.

The framework integrates:

- Structured preprocessing
- Class-conditional GAN training
- Controlled synthetic data generation
- Multi-experiment classifier training
- Statistical significance testing
- Clinical and representation-level analysis

This repository is designed as a research-grade experimental framework, not just a collection of training scripts.

## 2. Datasets

This project uses the following public datasets:

### HAM10000

- 10,015 dermoscopic images
- 7 diagnostic categories
- Strong class imbalance

Official source:  
https://www.kaggle.com/datasets/kmader/skin-cancer-mnist-ham10000

### ISIC 2018 (Task 3)

Used for evaluation and cross-dataset validation.

Official source:  
https://challenge.isic-archive.com/data/#2018

---

### Class Imbalance Motivation

Certain classes (e.g., AKIEC, DF, VASC) contain significantly fewer samples than dominant classes such as NV or MEL.

GAN-based augmentation is applied specifically to minority classes to:

- Increase minority class representation
- Improve macro-F1
- Improve minority recall
- Reduce class bias

## 4. Experimental Design

Three core classification experiments are conducted:

| Experiment | Description |
|------------|------------|
| exp_01_baseline | No augmentation |
| exp_02_base_aug | Standard augmentation (flip, rotate, etc.) |
| exp_03_gan_aug | GAN-based minority class augmentation |

Each experiment stores:

- config_snapshot.yaml
- Training logs
- metrics.json
- confusion_matrix.json
- classification_report.json
- predictions.csv

## 6. Pipeline

### Step 1 — Preprocessing

```
python main/run_01_preprocess.py
```

Includes:

- Hair artifact removal
- Lesion cropping
- Color constancy
- Resize & normalization
- Train/val/test splitting

---

### Step 2 — Train GAN & Generate Synthetic Images

```
python main/run_02_gan_generate.py
```

GAN is trained per minority class:

- AKIEC  
- DF  
- VASC  

Models implemented in:

```
src/gan/models/
```

---

### Step 3 — Train Classifiers

```
python main/run_03_all_train.py
```

Backbones:

- ResNet50
- EfficientNet-B0
- MobileNetV2

Training strategies:

- Baseline
- Basic augmentation
- GAN augmentation

---

### Step 4 — Evaluate

```
python main/run_04_all_evaluate.py
```

Outputs:

- Accuracy
- Precision / Recall / F1
- Per-class metrics
- Confusion matrix
- Calibration metrics

---

### Step 5 — Statistical Analysis

```
python main/run_05_analysis.py
```

Includes:

- Bootstrap confidence intervals
- McNemar test
- Significance testing
- Calibration analysis
- Embedding space analysis
- Clinical ABCD-inspired evaluation

## 7. Reproducibility

### 1. Clone repository

```
git clone 
cd DermGAN
```

### 2. Create virtual environment

```
python -m venv venv
```

Linux/Mac:
```
source venv/bin/activate
```

Windows:
```
.\venv\Scripts\activate
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

### 4. Run full pipeline

```

```

All experiments are controlled via YAML config files in:


Each experiment stores a `config_snapshot.yaml` for reproducibility.
